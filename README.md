# To-Do / Task Manager

## Project Description
This project is a simple task manager application written in Python. 
It allows users to create, manage, and organize tasks efficiently.

## Features
1. Add task
2. Edit task
3. Delete task
4. List all tasks
5. Mark task completed
6. Filter tasks by status
7. Filter tasks by priority
8. Add categories
9. Show upcoming tasks
10. Save/load tasks
11. Sort tasks by due date
12. Search tasks by keyword

## Feature Descriptions

Add Task  
Allows the user to create a new task with a title, priority, and due date.

Edit Task  
Allows the user to modify an existing task.

Delete Task  
Removes a selected task from the task list.

List All Tasks  
Displays all tasks stored in the system.

Mark Task Completed  
Marks a task as finished.

Filter by Status  
Displays tasks based on their completion status.

Filter by Priority  
Allows users to view tasks according to their priority level.

Add Categories  
Allows tasks to be grouped into categories.

Show Upcoming Tasks  
Displays tasks that are due soon.

Save / Load Tasks  
Stores tasks to a file and loads them when the program starts.

Sort by Due Date  
Sorts tasks based on their deadlines.

Search by Keyword  
Allows users to search tasks by keywords.
